# 🧠 PROJECT OVERVIEW

Build a modern, high-converting freelance website for:

👉 **AI Solution Maven**

This is NOT just a portfolio.

The website must:
- Showcase real products (especially PlanSight AI)
- Demonstrate technical capability
- Generate freelance leads
- Look like a professional SaaS product website

---

# 🎯 GOALS

1. Impress visitors with real working products
2. Clearly communicate services offered
3. Convert visitors into clients
4. Establish strong personal brand credibility

---

# 🧱 TECH STACK

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide React icons

---

# 🎨 DESIGN RULES (STRICT)

- Follow BRANDKIT.md exactly
- Use defined color palette and gradient
- Use Inter font (primary)
- Use consistent spacing (8pt grid)
- Use rounded UI (xl / 2xl)
- Use clean SaaS-style layout
- Hero section should use dark theme
- Content sections should use light theme

# ✍️ BRAND VOICE

- Clear
- Direct
- Practical
- No buzzwords
- Focus on outcomes, not technology

Example:
✅ "I build AI tools that help you make better decisions."
❌ "Leveraging advanced AI paradigms..."

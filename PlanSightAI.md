# PlanSight AI — Product Summary

## 1. Product Overview
PlanSight AI is an AI-powered project plan analyzer and sharing tool that allows users to upload, visualize, and understand project plans (e.g., MS Project files) with intelligent insights.

---

## 2. The Problem (Pain Points)

Project managers and stakeholders face several challenges when working with project plans:

- Project files (e.g., MS Project `.mpp`) are **complex and hard to understand**
- Stakeholders often **don’t have MS Project installed**, making sharing difficult
- Project data is **dense, technical, and not easily digestible**
- It’s hard to quickly identify:
  - Key tasks
  - Risks
  - Delays
- Existing tools focus on **editing and management**, not **clarity and communication**
- Sharing project plans with non-technical stakeholders is **friction-heavy**

👉 Result:
**Project plans become underutilized, misunderstood, and difficult to communicate.**

---

## 3. Core Value Proposition
**“Upload, visualize, share, and understand project plans with AI insights.”**

- Simplifies complex project plans  
- Provides interactive visualization (Gantt + task view)  
- Delivers AI-generated insights for better decision-making  
- Makes project plans easy to share with anyone  

---

## 4. Key Features (MVP Scope)

### Already Built
- Upload `.mpp` files  
- Task table (expand/collapse)  
- Gantt chart (days/weeks/months)  
- Column resizing  
- Sample project templates  

### High Priority Additions
- Shareable public links (no login required)  
- “My Tasks” filter (by assigned user)  
- AI-generated project summary  

---

## 5. Strategic Focus

PlanSight AI is **NOT**:
- A full project management system  
- A competitor to Asana, Monday, or MS Project  

PlanSight AI **IS**:
- A project visibility tool  
- A lightweight, fast viewer  
- An AI-enhanced insights platform  

---
